== Description ==

The plugin allows you to display posts from the current category in the sidebar.

= Docs & Support =

You can find more detailed information about plugin on [GitHub](https://github.com/gruz0/widget-to-display-posts-from-current-category).

== Installation ==

1. Download the plugin from wordpress.org
2. Upload it to your site (if you've installed it through Wordpress Admin Dashboard skip this step)
3. Activate
4. Appearence -> Widgets -> add the new widget
5. Save & Enjoy!
